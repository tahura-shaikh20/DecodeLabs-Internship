## Task 1: Rule-Based AI Chatbot

### Description
This project is a simple rule-based AI chatbot developed using Python. It responds to user inputs using predefined conditions and runs in a continuous loop.

### Features
- Handles greetings and basic conversation
- Provides date and time
- Responds using if-else logic
- Runs continuously until exit command

### How to Run
1. Open terminal or command prompt
2. Navigate to the project folder
3. Run the command:
   python task1.py
