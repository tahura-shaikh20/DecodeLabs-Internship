## Task 2: Data Classification using Machine Learning

### Description
This project involves building a basic classification model using a supervised machine learning algorithm. The model is trained on a dataset and used to predict the class of new data points.

### Dataset
The Iris dataset is used in this project. It contains measurements of different flower species, including:
- Sepal length
- Sepal width
- Petal length
- Petal width

The goal is to classify the flower into one of three species.

### Features
- Data loading and exploration
- Splitting dataset into training and testing sets
- Model training using Decision Tree Classifier
- Making predictions on test data
- Evaluating model performance using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report

### Technologies Used
- Python
- scikit-learn (sklearn)

### How to Run
1. Open terminal or command prompt
2. Navigate to the project folder
3. Install required library:
   pip install scikit-learn
4. Run the file:
   python task2.py

### Output
- Displays dataset information
- Shows training and testing split
- Prints predicted vs actual values
- Displays model accuracy
- Shows confusion matrix and classification report

### Learning Outcome
- Understanding supervised learning
- Building a classification model
- Evaluating model performance using multiple metrics


## Task 3: AI Recommendation System

### Description
This project implements a simple recommendation system based on user preferences. It uses similarity logic to match user interests with item features and provides personalized recommendations.

### Objective
To build a content-based recommendation engine using pattern matching and similarity techniques.

### Features
- Takes user input (interests)
- Matches input with item tags
- Uses Jaccard Similarity for comparison
- Displays similarity scores
- Provides top recommendations
- Highlights best match

### Technologies Used
- Python
- Set operations
- Basic AI logic

### How to Run
1. Open terminal or command prompt
2. Navigate to the project folder
3. Run the file:
   python task3.py
4. Enter your interests when prompted

### Example Input
python, ai

### Output
- Displays similarity scores for all items
- Shows top matching recommendations
- Displays best match

### Learning Outcome
- Understanding recommendation systems
- Applying similarity algorithms
- Implementing pattern matching logic
